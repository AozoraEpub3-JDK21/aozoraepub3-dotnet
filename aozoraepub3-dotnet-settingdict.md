## 1. 入力設定

---

### 1-1. 入力ファイル

| 項目 | 内容 |
|------|------|
| **英語名** | `InputFiles` |
| **UIコントロール** | ファイルリスト（ドラッグ＆ドロップ対応）＋「開く」ボタン |
| **デフォルト値** | なし |

**概要**

変換対象のファイルを指定する。対応形式は次の通り。

- `.txt` — 青空文庫形式テキストファイル（単体）
- `.zip` / `.txtz` — ZIP圧縮されたテキスト（＋画像）アーカイブ。圧縮内にテキストが複数ある場合はそれぞれ個別のEPUBとして変換される
- `.rar` — RARアーカイブ（SharpCompress使用）
- `.cbz` — 画像のみのZIPアーカイブ。**変換非対応のため警告を表示して処理をスキップ**する

複数ファイルを同時に指定すると順番に処理する。

---

### 1-2. 入力エンコード

| 項目 | 内容 |
|------|------|
| **英語名** | `InputEncoding` |
| **INIキー** | `-enc` |
| **UIコントロール** | ComboBox |
| **デフォルト値** | MS932 |
| **選択肢** | MS932 / UTF-8 |

**概要**

テキストファイルの文字エンコードを指定する。

青空文庫の古い作品はShift-JIS（MS932）で配布されていることが多い。UTF-8で配布されているファイル（Web小説ダウンロード等）を変換する場合はUTF-8を選択する。誤ったエンコードを指定すると文字化けが発生する。アーカイブ（ZIP/RAR）内のテキストにも同じエンコードが適用される。

---

### 1-3. ファイル名から表題・著者を使用

| 項目 | 内容 |
|------|------|
| **英語名** | `UseFileName` |
| **INIキー** | `-tf` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | OFF |

**概要**

ONにすると、ファイル名から `[著者名] タイトル` の形式で書誌情報を抽出し、テキスト本文中の表題・著者行より**優先**して使用する。

OFFのときは、ファイル名からの抽出値はテキスト本文中に表題・著者が見つからなかった場合の**補完用**として使う（上書きしない）。

ファイル名が `[夏目漱石] 坊っちゃん.txt` のような形式に限り機能する。

## 2. 書誌情報設定

---

### 2-1. 表題種別

| 項目 | 内容 |
|------|------|
| **英語名** | `TitleType` |
| **INIキー** | `-t` |
| **UIコントロール** | ComboBox |
| **デフォルト値** | 0（表題→著者名） |
| **選択肢** | 0〜4 |

**概要**

テキストファイルの冒頭に記載された書誌情報の並び順を指定する。青空文庫では先頭数行が「タイトル・著者名」になっているが、その並び順は作品ごとに異なるため、ここで指定する。

| 値 | 意味 |
|----|------|
| 0 | 1行目が表題、次行以降が著者名（最も一般的な青空文庫形式） |
| 1 | 1行目が著者名、次行以降が表題 |
| 2 | 表題→著者名だが、副題（ルビ付き原題等）を優先してタイトルに使用 |
| 3 | 表題のみ（著者名なし） |
| 4 | 書誌行なし（テキスト先頭から本文として扱う） |

誤った設定をすると、著者名がタイトルになったり書誌情報が本文扱いになったりする。

---

### 2-2. 表紙画像

| 項目 | 内容 |
|------|------|
| **英語名** | `CoverImage` |
| **INIキー** | `--cover / -c` |
| **UIコントロール** | ComboBox ＋ TextBox ＋ 参照ボタン |
| **デフォルト値** | 指定なし |
| **選択肢** | 指定なし / 先頭の挿絵 / 同名画像ファイル / ファイル指定 |

**概要**

EPUBの表紙に使用する画像を指定する。

- **指定なし** — 表紙画像なし
- **先頭の挿絵** (`0`) — テキスト中で最初に登場する画像（縦横64px超）を表紙として使用
- **同名画像ファイル** (`1`) — 入力ファイルと同じ名前で拡張子だけ `.png` / `.jpg` / `.jpeg` に変えたファイルを探して表紙にする
- **ファイル指定** — 任意の画像ファイルパスを指定。入力ファイルと同ディレクトリも自動検索する

---

### 2-3. 表紙ページ挿入

| 項目 | 内容 |
|------|------|
| **英語名** | `InsertCoverPage` |
| **INIキー** | `CoverPage` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | OFF |

**概要**

ONにすると、指定した表紙画像を使ってEPUBの冒頭に表紙ページ（独立したXHTMLページ）を挿入する。OFFにすると表紙画像はメタデータに設定されるのみで、独立ページとしては出力されない。

---

### 2-4. 表紙ページを目次に含める

| 項目 | 内容 |
|------|------|
| **英語名** | `InsertCoverPageToc` |
| **INIキー** | `CoverPageToc` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | OFF |

**概要**

表紙ページ挿入がONのとき、EPUBの目次（`nav.xhtml` および `toc.ncx`）に「表紙」エントリを追加するかどうかを制御する。ONにするとリーダーの目次メニューに「表紙」が表示されるようになる。

---

### 2-5. 表紙最大行数

| 項目 | 内容 |
|------|------|
| **英語名** | `MaxCoverLine` |
| **INIキー** | `MaxCoverLine` |
| **UIコントロール** | NumericUpDown |
| **デフォルト値** | 0（無制限） |

**概要**

「先頭の挿絵を表紙に使用」を選択したとき、何行目より後の画像は表紙として採用しないかを指定する。たとえば50を指定すると、50行目以降に初めて登場する画像は表紙に使われず、代わりに表紙なし扱いになる。本文が始まってから登場するカラーページや口絵などを表紙に誤用しないための制限。0は制限なし。

---

### 2-6. 表題ページ出力

| 項目 | 内容 |
|------|------|
| **英語名** | `TitlePageWrite` |
| **INIキー** | `TitlePageWrite` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | OFF |

**概要**

ONにすると、表題・著者名・出版社などを記した「表題ページ」を独立したXHTMLページとしてEPUBに挿入する。表題ページの見た目は「表題ページ種別」で制御する。OFFにすると表題ページは作られない。

---

### 2-7. 表題ページ種別

| 項目 | 内容 |
|------|------|
| **英語名** | `TitlePageType` |
| **INIキー** | `TitlePage` |
| **UIコントロール** | ComboBox |
| **デフォルト値** | 0（通常） |
| **有効条件** | 表題ページ出力がONのとき |
| **選択肢** | 0=通常 / 1=中央揃え / 2=横書き |

**概要**

表題ページのレイアウトテンプレートを選択する。

- **通常** — 縦書きレイアウトで表題・著者を並べる標準的なスタイル
- **中央揃え** — ページ縦横中央に表題・著者を配置するミドルスタイル
- **横書き** — 横書きテンプレートを使用（横書き設定時に用いる）

## 3. 出力設定

---

### 3-1. 出力先ディレクトリ

| 項目 | 内容 |
|------|------|
| **英語名** | `OutputDirectory` |
| **INIキー** | `--dst / -d` |
| **UIコントロール** | TextBox ＋ 参照ボタン |
| **デフォルト値** | 入力ファイルと同じディレクトリ |

**概要**

変換後のEPUBファイルを保存するディレクトリを指定する。空欄の場合は入力ファイルと同じディレクトリに出力される。セキュリティ上、指定ディレクトリ外へのパス指定は内部で拒否される。

---

### 3-2. 出力拡張子

| 項目 | 内容 |
|------|------|
| **英語名** | `OutputExtension` |
| **INIキー** | `-ext` |
| **UIコントロール** | ComboBox |
| **デフォルト値** | `.epub` |
| **選択肢** | `.epub` / `.kepub.epub` |

**概要**

出力ファイルの拡張子を指定する。KOBOリーダー向けに `.kepub.epub` 形式で出力したい場合はこちらを選択する。`.kepub.epub` は通常のEPUBと内容は同じだが拡張子が異なる。

---

### 3-3. 出力ファイル名を入力ファイル名に合わせる

| 項目 | 内容 |
|------|------|
| **英語名** | `UseInputFileName` |
| **INIキー** | `-of` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | OFF |

**概要**

ONにすると出力EPUBのファイル名が入力ファイルのベース名（拡張子を除いた部分）になる。OFFの場合は抽出した著者名・タイトルから `[著者名] タイトル.epub` というファイル名が自動生成される。著者名・タイトルが取得できなかった場合は入力ファイル名が使われる。

---

### 3-4. 対象端末

| 項目 | 内容 |
|------|------|
| **英語名** | `TargetDevice` |
| **INIキー** | `-device` |
| **UIコントロール** | ComboBox |
| **デフォルト値** | 汎用 |
| **選択肢** | 汎用 / Kindle |

**概要**

出力EPUBを最適化する対象端末を指定する。Kindleを選択すると `Epub3Writer` がKindle向けの調整（CSSや構造の変更）を適用する。汎用は標準的なEPUB3リーダー向け。

## 4. ページ構成設定

---

### 4-1. 縦書き / 横書き

| 項目 | 内容 |
|------|------|
| **英語名** | `Vertical` |
| **INIキー** | `-hor`（横書きフラグ） |
| **UIコントロール** | RadioButton（縦書き / 横書き） |
| **デフォルト値** | 縦書き |

**概要**

EPUBのテキスト組方向を指定する。縦書きを選ぶと `writing-mode: vertical-rl` のCSSが適用され、日本語の縦組みレイアウトになる。横書きは `writing-mode: horizontal-tb` で洋書スタイルになる。この設定は自動横組み（縦中横）など複数の変換処理にも影響する。

---

### 4-2. 目次ページ挿入

| 項目 | 内容 |
|------|------|
| **英語名** | `InsertTocPage` |
| **INIキー** | `TocPage` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | OFF |

**概要**

ONにすると、EPUB内にリーダー標準の目次（`nav.xhtml`）とは別に、テキストとして閲覧できる**独立した目次ページ**を挿入する。章タイトルをリンク付きで一覧表示するページが本文の前に追加される。

---

### 4-3. 目次縦書き

| 項目 | 内容 |
|------|------|
| **英語名** | `TocVertical` |
| **INIキー** | `TocVertical` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | OFF |
| **有効条件** | 目次ページ挿入がONのとき |

**概要**

挿入する目次ページのテキスト組方向を縦書きにするかどうかを指定する。本文が縦書きの場合に統一感を出すために使う。OFFにすると目次ページは横書きで出力される。

---

### 4-4. 表題を目次に含める

| 項目 | 内容 |
|------|------|
| **英語名** | `InsertTitleToc` |
| **INIキー** | `InsertTitleToc` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | ON |

**概要**

ONにすると、EPUB目次の先頭エントリとして作品タイトル（ファイル冒頭の表題行）をリンク付きで追加する。OFFにすると目次から表題エントリを省き、最初の章から目次が始まる。1冊1作品のシンプルな構成のときはOFFにするとすっきりした目次になる。

## 5. 文字・変換設定

---

### 5-1. 挿絵なし

| 項目 | 内容 |
|------|------|
| **英語名** | `NoIllust` |
| **INIキー** | `NoIllust` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | OFF |

**概要**

ONにすると、テキスト中の画像注記（`［＃（画像ファイル名）］` 等）をすべて無視し、画像を一切EPUBに含めない。表紙用に指定した画像のみは例外として含まれる。テキスト主体で読みたい場合やファイルサイズを最小化したい場合に使用する。画像がスキップされた旨はログに警告として記録される。

---

### 5-2. マーク用ID付与

| 項目 | 内容 |
|------|------|
| **英語名** | `WithMarkId` |
| **INIキー** | `MarkId` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | OFF |

**概要**

ONにすると、各段落の `<p>` タグに `id="kobo.行番号.1"` 形式の属性を付与する。KOBOなどのリーダーが栞・ハイライトの位置情報を精密に管理するために使用する。IDが増えるためファイルサイズがわずかに増加する。

---

### 5-3. 自動横組み

| 項目 | 内容 |
|------|------|
| **英語名** | `AutoYoko` |
| **INIキー** | `AutoYoko` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | ON |
| **有効条件** | 縦書きのとき |

**概要**

縦書きテキスト中に登場する数字・記号を自動的に横組み（縦中横）に変換する機能の大元のスイッチ。OFFにすると以下の4つの細分設定もすべて無効になる。ONにすると、サブ設定の内容に従って1桁・3桁の数字や記号を `縦中横` 注記で囲んで横向きに表示する。縦書き小説で「2024年」「100kg」などを自然に見せるために必要。

---

### 5-4. 数字1桁 自動横組み

| 項目 | 内容 |
|------|------|
| **英語名** | `AutoYokoNum1` |
| **INIキー** | `AutoYokoNum1` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | ON |
| **有効条件** | 自動横組みがONのとき |

**概要**

縦書き中に半角数字が1桁単独で登場したとき（前後が数字でないとき）に縦中横（横向き表示）へ変換する。「3分」「7人」など1文字の数字が自然な向きで読めるようになる。

---

### 5-5. 数字3桁 自動横組み

| 項目 | 内容 |
|------|------|
| **英語名** | `AutoYokoNum3` |
| **INIキー** | `AutoYokoNum3` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | ON |
| **有効条件** | 自動横組みがONのとき |

**概要**

半角数字が3桁連続する場合に縦中横へ変換する（例：`100`、`365`）。4桁以上は変換しない。

---

### 5-6. 英数字1桁 自動横組み

| 項目 | 内容 |
|------|------|
| **英語名** | `AutoYokoEQ1` |
| **INIキー** | `AutoYokoEQ1` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | ON |
| **有効条件** | 自動横組みがONのとき |

**概要**

縦書き中に `!` / `?` が1文字単独で現れた場合に縦中横へ変換する。「！」「？」として表示されることが多いが、半角感嘆符・疑問符を縦組みで自然に見せる。

---

### 5-7. 濁点種別

| 項目 | 内容 |
|------|------|
| **英語名** | `DakutenType` |
| **INIキー** | `DakutenType` |
| **UIコントロール** | ComboBox |
| **デフォルト値** | 1（CSSスタイル） |
| **選択肢** | 0=なし / 1=CSSスタイル / 2=外字フォント |

**概要**

縦書き中で文字の上に濁点・半濁点が重なる「合成文字」の処理方法を指定する。青空文庫では清音字＋濁点という2文字で表現されている文字（例：ウ゛→ヴ）の出力スタイルを制御する。

- **0 なし** — 濁点をそのまま後ろに並べて出力（標準的なUnicode処理）
- **1 CSSスタイル** — `<span class="dakuten">` でラップしてCSSで重ね合わせ表示する
- **2 外字フォント** — 外字フォントディレクトリに対応フォントがある場合はそれを使ってグリフ合成する

---

### 5-8. IVS BMP出力

| 項目 | 内容 |
|------|------|
| **英語名** | `IvsBMP` |
| **INIキー** | `IvsBMP` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | OFF |

**概要**

IVS（Ideographic Variation Sequence）のうち、BMP範囲（U+FE00〜U+FE0F）の異体字選択子を含む文字をEPUBに出力するかどうかを指定する。ONにすると、対応する外字フォントがない場合でもIVSセレクタを付けたまま出力する。OFFにするとIVSセレクタを除いて基底文字のみ出力する。対応EPUBリーダーが少ないため、一般的にはOFFでよい。

---

### 5-9. IVS SSP出力

| 項目 | 内容 |
|------|------|
| **英語名** | `IvsSSP` |
| **INIキー** | `IvsSSP` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | ON |

**概要**

IVSのうちSSP（Supplementary Special-purpose Plane）範囲（U+E0100〜U+E01EF）の異体字選択子を含む文字を出力するかどうかを指定する。青空文庫では JIS補助漢字などのルビ付き外字にこの範囲が使われる。ONにすると縦書き用の正立スタイルを付けてそのまま出力し、OFFにすると基底文字のみ出力する。

---

### 5-10. スペース禁則処理

| 項目 | 内容 |
|------|------|
| **英語名** | `SpaceHyphenation` |
| **INIキー** | `SpaceHyphenation` |
| **UIコントロール** | ComboBox |
| **デフォルト値** | 0（なし） |
| **選択肢** | 0=なし / 1=softspace / 2=en-space |

**概要**

縦書きテキスト中の全角スペースを禁則処理の折り返しポイントとして扱う方法を指定する。行の長さが均一でない縦書きレイアウトで、全角スペースが行頭・行末に来てしまうのを防ぐ。

- **0 なし** — 全角スペースをそのままUnicode文字として出力
- **1 softspace** — `<span class="fullsp"> </span>` に変換（CSSで幅調整）
- **2 en-space** — 全角スペースを `U+2000 U+2000`（enスペース2個）に変換

行の先頭で20文字を超えた位置にある単独の全角スペースのみが対象。

---

### 5-11. コメント出力

| 項目 | 内容 |
|------|------|
| **英語名** | `CommentPrint` |
| **INIキー** | `CommentPrint` |
| **UIコントロール** | CheckBox |
| **デフォルト値** | OFF |

**概要**

青空文庫テキストでは、50個以上の半角ハイフン（`--------------------------------------------------`）で囲まれたブロックを「コメント」（校正注・底本注記）として扱い、通常は無視する。ONにするとこのコメントブロックをEPUBに出力する。コメント変換（5-12）と組み合わせて使う。

---

### 5-12. コメント変換

| 項目 | 内容 |
